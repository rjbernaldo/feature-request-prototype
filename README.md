# Feature Request Prototype

Internal full stack prototype to handle incoming feature requests sent by clients.

Purpose of this project is to practice TDD and at the same time reinforce my knowledge of both old (relatively) and new tools for software development.

Built using [Node](https://github.com/nodejs/node), [Express](https://github.com/expressjs/express), [React](https://github.com/facebook/react), [React Bootstrap](https://github.com/react-bootstrap/react-bootstrap), [Mongoose](https://github.com/Automattic/mongoose), [Webpack](https://github.com/webpack/webpack), and [Jest](https://github.com/facebook/jest).

**Instructions**
```
$ npm install
$ npm build
$ npm start
```

**Tests**
```
npm test
```
